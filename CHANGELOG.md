# 变更记录

本项目在纳入 Git 前由 Cursor 协作迭代，以下为按对话顺序整理的功能演进（无法还原每一版文件快照，仅作历史说明）。

## 基础功能

- Fisher-Yates 乱序 + 按 N 抽取
- 号码解析（空格/逗号/中文标点）
- 深色主题、localStorage 历史

## 移动端与 UI

- 乱序池/抽取结果与号码列表统一展示，内容可滚动
- 底部固定操作栏（乱序抽取 + 抽取数量 stepper）
- 设置弹窗（抽取数、主题切换）
- 固定顶栏、卡片 header 数量统计
- textarea 风格号码展示（非 tag 列表）
- 乱序池 / 抽取结果 / 号码列表视觉区分（紫/青/默认）
- 全屏预览弹窗
- 结果卡片呼吸动画反馈（替代抽取成功 Toast）

## 抽取逻辑

- 优化目标：抽取结果**排序后**连号对数最少（非乱序池相邻）
- 修改抽取数量后，关闭设置弹窗时从同一乱序池重新抽取

## 复制与键盘

- Android/iOS 复制修复（copy 事件 + Clipboard API 回退 + 手动复制弹窗）
- 键盘 Enter / enterkeyhint 触发乱序抽取
- 抽取数量输入框支持 Android 全删重输

## 测试与工程化

- 抽取逻辑抽离至 `shuffle.js`
- Vitest 单元测试
- 修复 ES Module 在 `file://` 下无法加载导致按钮失效的问题
